# HitZero Flow

A criação de rotinas de cheerleading é uma atividade que exige conhecimento técnico, criatividade e domínio das regras competitivas. Entretanto, principalmente a nível competitivo, esse processo é complexo, pois deve equilibrar fluidez, segurança, dificuldade, habilidades do time e maximização da pontuação. Com o avanço da computação, torna-se possível utilizar algoritmos de otimização para automatizar a montagem dessas rotinas, dando aos treinadores uma base técnica em relação aos movimentos a serem executados.
Dessa forma, por meio da inserção da quantidade de pessoas no time, do nível competitivo e outras variáveis cruciais para a criação de uma rotina completa, o sistema será capaz de sugerir uma sequência de movimentos que formem uma coreografia completa e pontuável, reduzindo tempo de planejamento e permitindo que técnicos explorem novas possibilidades criativas. Assim, este trabalho busca unir esporte e tecnologia, trazendo uma solução inovadora para uma área pouco explorada academicamente.
